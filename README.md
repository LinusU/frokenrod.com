
# Website for Fröken Röd

## Configuration

Create the file `settings.json` and configure as follow:

```json
{
  "instagram": {
    "user_id": " ... ",
    "client_id": " ... ",
    "client_secret": " ... "
  },
  "gmail": {
    "user": " ... ",
    "pass": " ... "
  }
}
```

## Quick start

```sh
npm install
node .
```
